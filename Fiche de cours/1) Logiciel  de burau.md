## 👥 Contacts (créer / modifier)

* 📍 Barre de navigation → **Personnes** (ou **Contacts**)
* ➕ **Nouveau contact**
* ✏️ Pour modifier : double-clic sur le contact
* 🖼️ Photo : dans la fiche contact → zone **Ajouter une photo**
* 🗒️ Note : champ **Notes** dans la fiche contact
* 💬 “Messagerie instantanée” : dans la fiche contact → **Détails** / champs complémentaires (selon version)

---

## 👪 Groupe de contacts (liste de distribution)

* 📍 **Personnes / Contacts** → **Nouveau groupe de contacts**
* 🏷️ Nom : `Comité social`
* ➕ **Ajouter des membres** → depuis **Contacts Outlook** (ou Carnet d’adresses)
* ❌ Retirer un membre : sélectionner le membre dans la liste → **Supprimer**
* 💾 **Enregistrer & fermer**

---

## 🧾 Modèle Outlook (.oft)

* 📍 **Accueil** → **Nouveau courriel**
* Rédiger le message “type” (objet + corps + placeholders)
* 💾 **Fichier** → **Enregistrer sous**
* Type : **Modèle Outlook (*.oft)**
* 📁 Choisir le dossier demandé (ex. `Devoir-1`)

---

## 📇 vCard (envoyer des contacts)

* 📍 **Nouveau courriel** → onglet **Insertion**
* 🪪 **Carte de visite** → sélectionner le(s) contact(s)
* ✅ Ça insère des vCards en pièces jointes (format contact Outlook)

---

## 📅 Calendrier — rendez-vous, réunion, récurrence, rappel

### Rendez-vous (sans invités)

* 📍 **Calendrier** → **Nouveau rendez-vous**
* 🔁 Récurrence : **Récurrence…** (ou **Périodicité…**)
  → régler “Mensuel” + “3e vendredi” + heures
* ⏰ Rappel : menu **Rappel** (ex. 1 heure)

### Réunion (avec invités)

* 📍 **Calendrier** → **Nouvelle réunion**
* 👥 Ajouter des **participants** (À…)
* 📍 Ajouter **Lieu**, heure, durée

---

## ✅ Tâches — créer, récurrente, terminée, supprimer

* 📍 **Tâches** (ou “À faire / To Do” selon interface)
* ➕ **Nouvelle tâche**
* 🔁 Récurrence : bouton **Récurrence…** (hebdo / mensuel / etc.)
* 🔺 Priorité : **Haute / Normale / Basse**
* ✔️ Terminer : cocher **Terminée** (ou “Marquer comme terminée”)
* 🗑️ Supprimer : supprimer la tâche → va dans **Éléments supprimés**
* 🧹 Vider : clic droit **Éléments supprimés** → **Vider le dossier**

---

## 📨 Courriels — importance, catégories, CCI, transfert

* 🔺 Importance : dans un message → **Importance haute**
* 🏷️ Catégorie : message sélectionné → **Catégoriser** (renommer si besoin)
* 🔁 Transférer : **Transférer**
* 🕵️ CCI : fenêtre message → **Options** → afficher **Cci** (si masqué) → remplir le champ **Cci**

---

## 🗄️ Archivage & export PST

### Archivage automatique

* 📍 **Fichier** → **Options** → **Avancé** → **Paramètres d’archivage automatique…**
* Choisir dossier + durée + action (déplacer / supprimer selon consigne)

### Export PST

* 📍 **Fichier** → **Ouvrir et exporter** → **Importer/Exporter**
* **Exporter vers un fichier** → **Fichier de données Outlook (.pst)**
* Sélectionner **Boîte de réception** + cocher **Inclure les sous-dossiers**
* Nommer le fichier + choisir l’emplacement (clé USB)

---

## 🧠 Règles (classement auto)

* 📍 **Accueil** → **Règles** → **Gérer les règles et alertes**
* ➕ **Nouvelle règle**
* Condition : ex. “objet contient …”
* Action : **déplacer vers** dossier cible
* ✅ Activer / appliquer
