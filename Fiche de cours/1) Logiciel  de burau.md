<!-- ===================== 2) CONTACTS ===================== -->
## 👥 Contacts

- 📍 Barre de navigation → <b>Personnes</b>
- ➕ Créer : <b>Nouveau contact</b>
- ✏️ Modifier : double-clic sur le contact

<hr/>

<!-- ===================== 3) GROUPE DE CONTACTS ===================== -->
## 👪 Groupe de contacts

- 📍 <b>Personnes</b> → <b>Nouveau groupe de contacts</b>
- ➕ Membres : <b>Ajouter des membres</b> → <b>Contacts Outlook</b>
- ❌ Retirer : sélectionner → <b>Supprimer</b>

<hr/>

<!-- ===================== 4) MODELE OFT ===================== -->
## 🧾 Modèle (.oft)

- 📍 <b>Accueil</b> → <b>Nouveau courriel</b>
- 📍 <b>Fichier</b> → <b>Enregistrer sous</b> → type <b>Modèle Outlook (*.oft)</b>

<hr/>

<!-- ===================== 5) VCARD ===================== -->
## 📇 vCard

- 📍 <b>Nouveau courriel</b> → <b>Insertion</b> → <b>Carte de visite</b>

<hr/>

<!-- ===================== 6) CALENDRIER ===================== -->
## 📅 Calendrier

- 📍 Rendez-vous : <b>Calendrier</b> → <b>Nouveau rendez-vous</b> → <b>Récurrence…</b>
- 📍 Réunion : <b>Calendrier</b> → <b>Nouvelle réunion</b>

<hr/>

<!-- ===================== 7) TACHES ===================== -->
## ✅ Tâches

- 📍 <b>Tâches</b> → <b>Nouvelle tâche</b>
- 📍 Récurrence : bouton <b>Récurrence…</b>

<hr/>

<!-- ===================== 8) COURRIELS ===================== -->
## 📨 Courriels

- 📍 CCI : fenêtre message → <b>Options</b> → afficher <b>Cci</b>
- 📍 Catégorie : message sélectionné → <b>Catégoriser</b>

<hr/>

<!-- ===================== 9) ARCHIVAGE / EXPORT ===================== -->
## 🗄️ Archivage & export PST

- 📍 Archivage auto : <b>Fichier</b> → <b>Options</b> → <b>Avancé</b> → <b>Paramètres d’archivage automatique…</b>
- 📍 Export PST : <b>Fichier</b> → <b>Ouvrir et exporter</b> → <b>Importer/Exporter</b>

<hr/>

<!-- ===================== 10) REGLES ===================== -->
## 🧠 Règles

- 📍 <b>Accueil</b> → <b>Règles</b> → <b>Gérer les règles et alertes</b>
