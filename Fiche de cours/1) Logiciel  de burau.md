<!-- ===================== 2) CONTACTS ===================== -->
## 👥 Contacts (créer / modifier)

- 📍 Barre de navigation → <b>Personnes</b> (ou <b>Contacts</b>)
- ➕ <b>Nouveau contact</b>
- ✏️ Modifier : double-clic sur le contact
- 🖼️ Photo : zone <b>Ajouter une photo</b>
- 🗒️ Note : champ <b>Notes</b>
- 💬 Messagerie instantanée : onglet/zone <b>Détails</b> (selon version)

<hr/>

<!-- ===================== 3) GROUPE DE CONTACTS ===================== -->
## 👪 Groupe de contacts (liste de distribution)

- 📍 <b>Personnes / Contacts</b> → <b>Nouveau groupe de contacts</b>
- 🏷️ Nom : <code>Comité social</code>
- ➕ <b>Ajouter des membres</b> → depuis <b>Contacts Outlook</b> (ou carnet d’adresses)
- ❌ Retirer : sélectionner le membre → <b>Supprimer</b>
- 💾 <b>Enregistrer & fermer</b>

<hr/>

<!-- ===================== 4) MODELE OFT ===================== -->
## 🧾 Modèle Outlook (.oft)

- 📍 <b>Accueil</b> → <b>Nouveau courriel</b>
- 🧱 Rédiger le message “type” (objet + corps + placeholders)
- 💾 <b>Fichier</b> → <b>Enregistrer sous</b>
- 🧩 Type : <b>Modèle Outlook (*.oft)</b>
- 📁 Choisir le dossier demandé (ex. <code>Devoir-1</code>)

<hr/>

<!-- ===================== 5) VCARD ===================== -->
## 📇 vCard (envoyer des contacts)

- 📍 <b>Nouveau courriel</b> → onglet <b>Insertion</b>
- 🪪 <b>Carte de visite</b> → sélectionner le(s) contact(s)
- ✅ Outlook ajoute les vCards en pièces jointes

<hr/>

<!-- ===================== 6) CALENDRIER ===================== -->
## 📅 Calendrier — rendez-vous, réunion, récurrence, rappel

### Rendez-vous (sans invités)
- 📍 <b>Calendrier</b> → <b>Nouveau rendez-vous</b>
- 🔁 <b>Récurrence…</b> / <b>Périodicité…</b> → régler “Mensuel” + “3e vendredi” + heures
- ⏰ <b>Rappel</b> : choisir (ex. 1 heure)

### Réunion (avec invités)
- 📍 <b>Calendrier</b> → <b>Nouvelle réunion</b>
- 👥 Ajouter des participants (champ <b>À…</b>)
- 📍 Renseigner <b>Lieu</b>, heure, durée

<hr/>

<!-- ===================== 7) TACHES ===================== -->
## ✅ Tâches — créer, récurrente, terminée, supprimer

- 📍 <b>Tâches</b> (ou “À faire / To Do” selon interface)
- ➕ <b>Nouvelle tâche</b>
- 🔁 <b>Récurrence…</b> (hebdo / mensuel / etc.)
- 🔺 <b>Priorité</b> : Haute / Normale / Basse
- ✔️ Terminer : cocher <b>Terminée</b> (ou “Marquer comme terminée”)
- 🗑️ Supprimer : la tâche part dans <b>Éléments supprimés</b>
- 🧹 Vider : clic droit <b>Éléments supprimés</b> → <b>Vider le dossier</b>

<hr/>

<!-- ===================== 8) COURRIELS ===================== -->
## 📨 Courriels — importance, catégories, CCI, transfert

- 🔺 Importance : dans un message → <b>Importance haute</b>
- 🏷️ Catégorie : sélectionner le message → <b>Catégoriser</b> (renommer si besoin)
- 🔁 Transférer : bouton <b>Transférer</b>
- 🕵️ CCI : fenêtre message → onglet <b>Options</b> → afficher <b>Cci</b> → renseigner le champ

<hr/>

<!-- ===================== 9) ARCHIVAGE / EXPORT ===================== -->
## 🗄️ Archivage & export PST

### Archivage automatique
- 📍 <b>Fichier</b> → <b>Options</b> → <b>Avancé</b> → <b>Paramètres d’archivage automatique…</b>
- 🧭 Choisir dossier + durée + action (déplacer / supprimer selon consigne)

### Export PST
- 📍 <b>Fichier</b> → <b>Ouvrir et exporter</b> → <b>Importer/Exporter</b>
- 📤 <b>Exporter vers un fichier</b> → <b>Fichier de données Outlook (.pst)</b>
- ✅ Sélection : <b>Boîte de réception</b> + cocher <b>Inclure les sous-dossiers</b>
- 💾 Nommer + choisir l’emplacement (clé USB)

<hr/>

<!-- ===================== 10) REGLES ===================== -->
## 🧠 Règles (classement auto)

- 📍 Onglet <b>Accueil</b> → <b>Règles</b> → <b>Gérer les règles et alertes</b>
- ➕ <b>Nouvelle règle</b>
- 🔎 Condition : ex. “objet contient …”
- 📁 Action : <b>déplacer vers</b> dossier cible
- ✅ Activer / appliquer
