# 🧠 Fiche mémo — Word (cours) : “où cliquer” (Devoir 2 — chapitres 7 à 13)

<hr/>

## 🗂️ Fichiers / Enregistrement / PDF
- 📍 Ouvrir : **Fichier** → **Ouvrir**
- 📍 Enregistrer sous : **Fichier** → **Enregistrer sous**
- 📍 Export PDF : **Fichier** → **Enregistrer sous** → Type **PDF** (ou **Fichier** → **Exporter** → **Créer PDF/XPS**)

<hr/>

## 📝 Collage “en gardant la mise en forme du document”
- 📍 Coller : **Accueil** → (icône) **Coller** ▼
- ✅ Choix rapide : **Fusionner la mise en forme** / **Conserver le texte uniquement**
- ⌨️ Variante : **Ctrl+Alt+V** (Collage spécial) → **Texte sans mise en forme**

<hr/>

## ✒️ Corrections visibles (barré + surlignage)
- 📍 Barré : **Accueil** → groupe **Police** → **Barré**
- 📍 Surlignage : **Accueil** → **Couleur de surlignage du texte**
- 📍 Retirer surlignage : **Couleur de surlignage** → **Aucune couleur**

<hr/>

## 🔗 Espaces insécables / traits d’union insécables (pour éviter la coupure de ligne)
- ⌨️ Espace insécable : **Ctrl+Maj+Espace**
- ⌨️ Trait d’union insécable : **Ctrl+Maj+-**

<hr/>

## ✅ Révision (orthographe/grammaire + options “volet Vérification”)
- 📍 Lancer : **Révision** → **Vérification** → **Orthographe et grammaire**
- 📍 “Mot répété” : dans le volet → action **Supprimer** (ou “Modifier”) selon la proposition
- 📍 Ajouter un mot au dictionnaire : dans le volet → **Ajouter au dictionnaire**
- 📍 Auto-correction majuscules : **Fichier** → **Options** → **Vérification** → **Options de correction automatique…**
  → activer **Mettre une majuscule en début de phrase**

<hr/>

## 💬 Commentaires + Suivi des modifications
- 📍 Nouveau commentaire : **Révision** → **Nouveau commentaire**
- 📍 Suivi des modifs : **Révision** → **Suivi** → **Suivi des modifications** (Activer)
- 📍 Affichage : **Révision** → **Suivi** → **Affichage pour révision** (Simple/Toutes les marques)

<hr/>

## 📌 Puces / Numérotation / Retraits (liste plus à droite)
- 📍 Numérotation : **Accueil** → **Numérotation**
- 📍 Décaler la liste à droite : **Accueil** → **Augmenter le retrait**
- 📍 Ajuster finement : clic droit sur la liste → **Ajuster les retraits de la liste**

<hr/>

## 🖼️ Images / Logo (insertion + placement)
- 📍 Insérer image : **Insertion** → **Images** → **Cet appareil**
- 📍 Habillage : cliquer l’image → **Format de l’image** → **Habillage du texte** (Carré / Rapproché)
- 📍 Position : **Format de l’image** → **Position** (ou déplacer à la souris)
- 📍 Rogner : **Format de l’image** → **Rogner**

<hr/>

## 📏 Tabulations (aligner une année à droite)
- 📍 Définir taquet : **Accueil** → groupe **Paragraphe** (petite flèche) → **Tabulations…**
  → Position **16 cm** → Alignement **Droite** → **Définir**

<hr/>

## 🧩 Tableaux (styles, tri, colonnes égales, P1/P2, en-tête répété)
- 📍 Insérer : **Insertion** → **Tableau**
- 📍 Style “1 ligne sur 2” : cliquer le tableau → **Création de tableau** → **Styles de tableau** (lignes en bande)
- 📍 Trier : **Disposition (Outils de tableau)** → **Trier** (Nom, puis Prénom)
- 📍 Supprimer ligne/colonne : **Disposition** → **Supprimer** → (Lignes/Colonnes)
- 📍 Colonnes égales : **Disposition** → **Distribuer les colonnes**
- 📍 Lignes égales : **Disposition** → **Distribuer les lignes**
- 📍 Fusionner / fractionner : **Disposition** → **Fusionner les cellules** / **Fractionner les cellules**
- 📍 Répéter ligne de titres : sélectionner la 1re ligne → **Disposition** → **Répéter les lignes d’en-tête**
- 📍 Centrer le tableau dans la page : clic droit tableau → **Propriétés du tableau** → **Alignement : Centré**

<hr/>

## 🖨️ Impression (Backstage) — options demandées
- 📍 Ouvrir : **Fichier** → **Imprimer**
- 📍 Pages par feuille : **Paramètres** → **1 page par feuille** ▼ → choisir **6 pages par feuille**
- 📍 Imprimer pages 1 et 5 : **Pages :** `1,5`
- 📍 Nombre de copies : champ **Copies** (ex. 2)

<hr/>

## 🔒 Protection (formulaire) + mot de passe
- 📍 Protéger : **Révision** → **Protéger** → **Restreindre la modification**
- ✅ Autoriser : **Remplissage de formulaires**
- 🔐 Activer : **Oui, activer la protection** → mot de passe (ex. `Form123`)

<hr/>

## 📐 Mise en page (marges, orientation, taille, bordure, alignement vertical)
- 📍 Marges / Orientation / Taille : **Mise en page** → **Marges** / **Orientation** / **Taille (Légal)**
- 📍 Appliquer à une seule page : **Mise en page** → **Sauts** → **Saut de section (Page suivante)**
  puis **Mise en page** → (marges/orientation/taille) → **Appliquer à : Cette section**
- 📍 Bordure de page : **Conception** → **Bordures de page** → **Encadrement** → style **Ligne double**
- 📍 Alignement vertical (page de présentation) :
  **Mise en page** → (petite flèche “Mise en page”) → onglet **Disposition** → **Alignement vertical : Justifié**

<hr/>

## 🔢 Numéros de page + sections (roman / arabe / “Page X sur Y”)
![WhatsApp Image 2026-01-15 at 13 52 54](https://github.com/user-attachments/assets/feb748d9-0262-4403-aaab-186e3b94a026)

- 📍 Numéro en haut à droite : **Insertion** → **Numéro de page** → **Haut de page** → **Droite**
- 📍 Format (i, ii…) : **Insertion** → **Numéro de page** → **Format des numéros de page…**
- 📍 Redémarrer à 1 : **Format des numéros…** → **À partir de : 1**
- 📍 Cacher le numéro sur la 1re page d’une section :
  double-clic en en-tête/pied → **Outils En-tête et pied de page** → **Première page différente**
- 📍 “Page X sur Y” : **Insertion** → **Numéro de page** → choisir un modèle **Page X sur Y**
- 📍 Pied de page texte gauche/droite (sur une plage de pages) :
  double-clic pied de page → désactiver **Lier au précédent** (si besoin)
  → écrire à gauche / appuyer **Tab** (ou alignement droite) pour la zone droite


# 🧠 Fiche mémo — Excel (cours) : “où cliquer” (Devoir 3 — chapitres 14 à 19)

<hr/>

## 💾 Ouvrir / Enregistrer sous
- 📍 Ouvrir : **Fichier** → **Ouvrir**
- 📍 Enregistrer sous (renommer le classeur) : **Fichier** → **Enregistrer sous**
- 📍 Export PDF (si demandé) : **Fichier** → **Enregistrer sous** → Type **PDF**

<hr/>

## 🧹 Effacer contenu (sans effacer formats)
- 📍 Sélection cellule/plage → **Accueil** → **Effacer** ▼ → **Effacer le contenu**

<hr/>

## 🏷️ Nommer une cellule / une plage (ex. Salaire_base)
- 📍 Sélectionner la cellule (ex. C3) → zone **Nom** (à gauche de la barre de formule) → taper le nom → **Entrée**
  (ou) **Formules** → **Définir un nom**

<hr/>

## 🧮 Formules / fonctions (saisie + recopie)
- 📍 Saisir formule : cliquer la cellule → barre de formule → **Entrée**
- 📍 Recopier vers le bas/droite : poignée de recopie (petit carré en bas à droite de la cellule)
- 📍 Fonctions rapides : **Formules** → **Somme automatique** ▼ (Somme, Moyenne, Min, Max, Nb)

<hr/>

## 🔢 Formats de nombres (monétaire, séparateur, décimales, dates)
- 📍 **Accueil** → groupe **Nombre**
  - **Monétaire** / **Comptabilité**
  - **Séparateur de milliers**
  - **Augmenter/Réduire les décimales**
- 📍 Format Date : **Accueil** → **Nombre** ▼ → **Date**

<hr/>

## ✍️ Ajuster colonnes / lignes / renvoi à la ligne
- 📍 Ajuster largeur : double-clic sur la limite d’en-tête (A|B), ou **Accueil** → **Format** → **Ajuster la largeur**
- 📍 Ajuster hauteur : **Accueil** → **Format** → **Ajuster la hauteur**
- 📍 Renvoi automatique : **Accueil** → **Renvoyer à la ligne automatiquement**
- 📍 Saut de ligne manuel dans une cellule : **Alt+Entrée**

<hr/>

## 🧾 En-tête / pied de page (titre + date du jour)
- 📍 **Insertion** → **Texte** → **En-tête et pied de page**
- 📍 Zone gauche/droite : cliquer la zone → insérer texte
- 📍 Code date : onglet **Création (Outils En-tête/Pied)** → **Date actuelle**

<hr/>

## 🧱 Insérer / supprimer lignes & colonnes
- 📍 Insérer colonne : clic droit lettre de colonne → **Insérer**
- 📍 Insérer ligne : clic droit numéro de ligne → **Insérer**
- 📍 Supprimer : clic droit → **Supprimer**
- 📍 Copier les mises en forme : **Accueil** → **Reproduire la mise en forme** (pinceau)

<hr/>

## 🖨️ Mise en page / impression (paysage, échelle, centrer, marges, quadrillage)
- 📍 **Mise en page**
  - **Orientation** → **Paysage**
  - **Mise à l’échelle** → **Largeur : 1 page** (et/ou **Hauteur : 1 page** si demandé)
  - **Marges** → **Marges personnalisées** → cocher **Centrer sur la page : Horizontalement**
  - **Marges** → régler **Marge supérieure** (ex. 3 cm)
  - **Options de feuille** → **Quadrillage** → **Imprimer** (activer/désactiver)
- 📍 Aperçu : **Fichier** → **Imprimer**

<hr/>

## 🧾 Répéter la ligne des titres à l’impression
- 📍 **Mise en page** → **Imprimer les titres**
- 📍 Champ **Lignes à répéter en haut** → sélectionner la ligne d’en-têtes → **OK**

<hr/>

## 📏 Largeur identique des colonnes (ex. 12)
- 📍 Sélectionner colonnes concernées → **Accueil** → **Format** → **Largeur de colonne…** → saisir **12**

<hr/>

## 🔻 Sauts de page (déplacer un saut vers le bas)
- 📍 **Affichage** → **Aperçu des sauts de page**
- 📍 Glisser la ligne bleue (saut) **d’une ligne vers le bas**
- Retour vue normale : **Affichage** → **Normal**

<hr/>

## 🙈 Masquer / afficher des colonnes
- 📍 Masquer : sélectionner colonnes → clic droit → **Masquer**
- 📍 Afficher : sélectionner les colonnes autour → clic droit → **Afficher**

<hr/>

## 🧠 Fonctions utiles du devoir (raccourcis “quoi mettre”)
- 📍 Date du jour : `AUJOURDHUI()`
- 📍 Mois (numéro) : `MOIS(date)`
- 📍 Âge / années (logique) : `DATEDIF(date_naissance;AUJOURDHUI();"Y")` (si dispo) / sinon logique via YEAR()
- 📍 Entier inférieur (pas de décimales) : `ENT(valeur)`
- 📍 Moyenne conditionnelle : `MOYENNE.SI(plage_critere;critere;plage_moyenne)` (ou `MOYENNE.SI.ENS`)
- 📍 Compter conditionnel : `NB.SI(plage;critere)` (ou `NB.SI.ENS`)
- 📍 Arrondir à 1 décimale : `ARRONDI(valeur;1)`
- 📍 Condition double : `SI(ET(cond1;cond2);"Oui";"Non")`

<hr/>

## 📊 Graphiques — secteur + histogramme + modifs avancées
### Secteur (camembert) + nouvelle feuille
- 📍 Sélectionner données → **Insertion** → **Graphiques** → **Secteurs**
- 📍 Style / couleurs : **Création de graphique** → **Styles** / **Modifier les couleurs**
- 📍 Légende à droite : bouton **+** (Éléments du graphique) → **Légende** → **Droite**
- 📍 % sur bords extérieurs : **+** → **Étiquettes de données** → **Autres options** → **Pourcentage** + **Position : Bord extérieur**
- 📍 Excentrer un secteur : cliquer le secteur → glisser légèrement vers l’extérieur (ou **Format du point** → **Explosion**)
- 📍 Nouvelle feuille : **Création de graphique** → **Déplacer le graphique** → **Nouvelle feuille**

### Histogramme groupé
- 📍 Sélectionner données → **Insertion** → **Histogramme** → **Histogramme groupé**
- 📍 Titre/axes/légende : bouton **+** → **Titres des axes** / **Légende (Bas)** / **Titre du graphique**
- 📍 Axe vertical en Nombre 0 décimale : clic droit sur l’axe → **Format de l’axe** → **Nombre** → **Décimales : 0**

### Modifier la source / intervertir / type de série (combo)
- 📍 **Création de graphique** → **Sélectionner des données** (retirer/ajouter séries, années, colonnes)
- 📍 **Création de graphique** → **Intervertir ligne/colonne**
- 📍 Changer type d’une série : clic droit série → **Modifier le type de graphique des séries…** (Combo) → choisir **Courbe avec marqueurs**
- 📍 Supprimer titre axe : cliquer le titre → **Suppr**

<hr/>

## 📥 Import fichier Texte/CSV + figer + protéger
- 📍 Import : **Données** → **À partir du texte/CSV** → choisir fichier → **Charger**
- 📍 Insérer 3 lignes en haut : clic droit **ligne 1** → **Insérer** (x3)
- 📍 Figer 1re colonne : **Affichage** → **Figer les volets** → **Figer la première colonne**
- 📍 Protéger feuille : **Révision** → **Protéger la feuille** → mot de passe (ex. `recolte`)

<hr/>

## 🧩 Macros (enregistrement, stockage, exécution, .xlsm)
- 📍 Onglet Développeur (si absent) : **Fichier** → **Options** → **Personnaliser le ruban** → cocher **Développeur**
- 📍 Enregistrer macro : **Développeur** → **Enregistrer une macro**
  - Stocker dans : **Ce classeur** (si la macro doit suivre le fichier)
  - Raccourci : ex. **Ctrl+Maj+T**
- 📍 Actions typiques à enregistrer (mise en forme/mise en page) :
  - Largeur colonnes A:H : sélectionner A:H → **Accueil** → **Format** → **Largeur de colonne…**
  - Paysage : **Mise en page** → **Orientation** → **Paysage**
  - Tenir sur 1 page : **Mise en page** → **Mise à l’échelle** → **Largeur 1 page**
  - Centrer horizontalement : **Mise en page** → **Marges** → **Marges personnalisées** → **Centrer : Horizontalement**
- 📍 Exécuter macro : **Développeur** → **Macros** → **Exécuter** (ou via raccourci)
- 📍 Enregistrer le fichier avec macros : **Fichier** → **Enregistrer sous** → type **Classeur Excel prenant en charge les macros (*.xlsm)**

<hr/>

## 🔎 Filtrer + masquer colonnes (pour macros “listes”)
- 📍 Filtrer : **Données** → **Filtrer**
- 📍 Filtre “Oui” : flèche du filtre sur la colonne → cocher uniquement **Oui**
- 📍 Masquer colonnes : sélectionner colonnes → clic droit → **Masquer**
- 📍 Macro “Rétablir” : enlever filtres + réafficher colonnes (clic droit → **Afficher**)
