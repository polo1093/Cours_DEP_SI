Un **pronom relatif** relie 2 idées et introduit une **proposition relative** pour préciser un nom.

## Les principaux pronoms relatifs (EN → FR + usage)

### **who** (personnes) → *qui*

* Sujet (souvent) / courant
* *The technician **who** helped me…* = Le technicien **qui** m’a aidé…

### **whom** (personnes, objet) → *que / à qui*

* Plus **formel**, surtout après une préposition
* *To **whom** did you speak?* = À **qui** avez-vous parlé ?
* En anglais courant, on remplace souvent par **who** : *Who did you speak to?*

### **whose** (possession) → *dont / duquel*

* Personnes **et** choses
* *The client **whose** account is blocked…* = Le client **dont** le compte est bloqué…

### **which** (choses / animaux) → *qui / lequel*

* Très courant après une **virgule** (info additionnelle)
* *The printer, **which** is new, is already broken.* = L’imprimante, **qui** est neuve, est déjà en panne.

### **that** (personnes ou choses) → *qui/que*

* Très fréquent en anglais “neutre” **sans virgule** (info essentielle)
* *The email **that** I sent…* = L’email **que** j’ai envoyé…

### **where** (lieu) → *où*

* *The office **where** I work…* = Le bureau **où** je travaille…

---

## Point clé : **virgule ou pas ?** 🧠

* **Sans virgule** = info **essentielle** (restrictive) → souvent **that / who / which**

  * *The file that you sent…* (c’est ce fichier précis)
* **Avec virgule** = info **bonus** (non-essentielle) → plutôt **which / who**, **pas that**

  * *The file, which you sent yesterday, …*

---

## Raccourci “bureau”

* Personne : **who**
* Objet/outil/fichier : **that** (souvent) ou **which**
* Possession : **whose**
* Lieu : **where**
* Très formel : **whom** (surtout après préposition)
